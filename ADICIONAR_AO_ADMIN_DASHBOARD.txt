═══════════════════════════════════════════════════════════════
📝 COMO USAR O COMPONENTE DE VERIFICAÇÃO
═══════════════════════════════════════════════════════════════

OPÇÃO 1: Adicionar ao Admin Dashboard
───────────────────────────────────────────────────────────────

1. Abra: src/pages/admin/AdminDashboard.jsx

2. Adicione o import:
   import { AdminCheckSubscription } from '@/components/admin/AdminCheckSubscription';

3. Adicione após os cards de estatísticas:
   <div className="mt-8">
     <AdminCheckSubscription />
   </div>

OPÇÃO 2: Criar página separada
───────────────────────────────────────────────────────────────

Criar: src/pages/admin/AdminVerificarAssinatura.jsx

E adicionar rota no App.jsx:
<Route path="verificar-assinatura" element={<AdminVerificarAssinatura />} />

═══════════════════════════════════════════════════════════════
🎯 O QUE O COMPONENTE FAZ
═══════════════════════════════════════════════════════════════

✅ Permite inserir User ID e Email
✅ Verifica assinatura via API
✅ Mostra resultado detalhado:
   - Tem acesso?
   - É assinante?
   - Está em trial?
   - Status completo
   - Datas de expiração
   - Dias restantes (trial)
✅ Mostra JSON completo (expandível)

═══════════════════════════════════════════════════════════════
💡 CASOS DE USO
═══════════════════════════════════════════════════════════════

1. Verificar assinatura de usuário que reportou problema
2. Validar se usuário tem acesso antes de dar suporte
3. Verificar status antes de cancelar/renovar
4. Debug de problemas de acesso
