# 🚀 Quick Start: n8n no Render

## ⚡ Resumo Rápido

1. ✅ Conectar repositório no Render
2. ✅ Render detectará `render.yaml` automaticamente
3. ✅ Configurar variáveis de ambiente no painel
4. ✅ Deploy automático
5. ✅ Acessar n8n e criar workflow

---

## 📚 Documentação Completa

Siga o guia completo: **`GUIA_N8N_RENDER.md`**

---

## 🔑 Variáveis de Ambiente Necessárias

No painel do Render, configure:

1. **N8N_BASIC_AUTH_PASSWORD** - Senha para login (gerar uma senha forte)
2. **SUPABASE_URL** - `https://wwwwyuwighdehmvnolrl.supabase.co`
3. **SUPABASE_SERVICE_ROLE_KEY** - Sua service role key (secreta!)

**As outras variáveis já estão configuradas no `render.yaml`!**
