PROMPT PARA REDESIGN VISUAL - LWDigitalForge

Você está trabalhando no redesign visual completo do site da LWDigitalForge (https://github.com/Aldebaran-LW/LWDigitalForge_Texte). O site já possui todas as funcionalidades implementadas, mas precisa de um visual moderno, atual (2025), profissional, com visualizações e micro-interações.

STACK: React 18 + Vite + Tailwind CSS + Framer Motion + Radix UI + Supabase + Mercado Pago

FUNCIONALIDADES QUE DEVEM SER MANTIDAS (NÃO ALTERAR LÓGICA):

1. Sistema de Tema Dark/Light - Toggle no header, persistência, transições suaves
2. Carrinho de Compras - Sidebar deslizante, adicionar/remover, ajustar quantidades, checkout Mercado Pago, badge contador
3. Animações Framer Motion - Header entrada, hero escalonado, cards hover, scroll animations, menu mobile, cart slide-in
4. Navegação - Menu desktop/mobile, hover states, navegação condicional
5. Produtos - Grid responsivo, cards clicáveis, preços (mensal/anual/vitalício), detail page com seleção planos, teste grátis
6. Autenticação - Login/Cadastro Supabase, OAuth Google, recuperação senha, proteção rotas, exibir usuário logado
7. Toast Notifications - Feedback visual, variantes

COMPONENTES PRINCIPAIS:
- Header, Footer, HeroSection, ProductsSection, BenefitsSection
- ProductCard, ProductsList, ShoppingCart
- ProductDetailPage, PaginaProdutos, HomePage
- AdminLayout, AdminDashboard, AdminGerenciarProdutos, AdminFormularioProduto, AdminVendas, AdminUsuarios
- PortalMeusProdutos, PortalTestes, PortalPagamentos
